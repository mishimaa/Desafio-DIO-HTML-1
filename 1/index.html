Informações sobre as Tags

Todas as tags começam com "<" e terminam com ">". Para fechar ela é necessário utilizar a barra "/".
Toda página HTML consiste nas tags "html", "head" e "body". Tudo que está dentro do body é o que de fato aparece ao usuário.
O que fica dentro da tag "head" são informações sobre a página, metadados, e o que é pré-carregado antes de ser exibido ao usuário.

Tags

Algumas tags aprendidas nesta aula foram o "i", que deixa o texto em itálico e o "strong" que deixa o texto em negrito.
É possível utilizar essas tags juntas, deixan o texto em itálico e em negrito, ou só em uma palavra em específica..

Sobre os Atributos

Atributos são propriedades definidas dentro de uma tag, e que modificam o funcionamento dela. Existem atributos mais específicos e outros mais genéricos.
Um exemplo seria a tag "style", onde é possível modificar a tag com CSS dentro do próprio código HTML.
Por meio dos atributos e tags, é possível modificar imagens, campos de texto, formulários, entre várias outras coisas.

Listas Ordenadas e Não Ordenadas

(voltar)

Listas Ordenadas

As listas ordenadas são listas que o navegador interpreta como sendo sequenciais, apresentada em ordem númerica.
A tag utilizada é a "ol", onde é necessário também ser usada a tag filha "li" para listar cada item.

Listas Não Ordenadas

As listas não ordenadas são listas que não possuem ordem específica.
A tag utilizada é a "ul", onde também é necessário a utilização da tag filha "li" para listar cada item.

Em ambos os casos é possível criar listas dentro de outras listas.

Criando Links no HTML

Para criar links no código HTML, é preciso utilizar a tag "a", com um atributo chamado "href" (que é onde vai o endereço linkado).
Um atributo importante também é o "target". Com ele é possível fazer com que o link seja aberto em outra aba ou outra janela do navegador.
E com o atributo "title", é possível dar uma descrição melhor sobre o link, fazendo com que apareça um balão com texto quando o usuário passa o mouse em cima do link.
Além de fazer links para outros sites, também é possível fazer links para outras páginas do próprio site. Além de links para partes específicas da própria página.
